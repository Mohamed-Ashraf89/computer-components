# computer-components# 🖥️ Computer Components Web Project

A fully responsive, semantic, and well-structured two-page website built as part of the **Coursera Web Development Foundations** assignment. This project serves as an educational guide to understanding how computer hardware works, diving into the core internal components that power modern computing.

## 📌 Project Requirements Achieved

| Requirement | Description | Status |
| :--- | :--- | :---: |
| **Valid HTML5 Structure** | Features fully formed pages with a strict `<!DOCTYPE html>`, `<head>`, and `<body>`. | ✅ Passed |

| **Lists Implementation** | Includes ordered lists (`<ol>`) for execution steps and unordered lists (`<ul>`) for features. | ✅ Passed |
